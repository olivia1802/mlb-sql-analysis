# MLB SQL Analysis

This project explores offensive performance trends across Major League Baseball teams and players using SQL queries applied to 2023 batting statistics.

## Project Goals

The objective of this project was to investigate how offensive production varies across MLB teams and players and to identify factors associated with strong offensive performance beyond home run production alone.

## Research Questions

* Which players ranked among the league leaders in home runs?
* Which teams demonstrated the strongest offensive production based on average OPS?
* How do team home run totals compare with average offensive production?
* Which teams demonstrated the strongest plate discipline based on walk-to-strikeout ratio?
* Which players generated strong offensive production despite relatively low home run totals?

## Key Findings

* Teams with the highest home run totals did not necessarily record the highest average OPS.
* MIL and PHI ranked among the league leaders in average OPS despite producing fewer home runs than several other teams.
* SDP and LAD recorded the strongest walk-to-strikeout ratios, suggesting stronger plate discipline.
* Players such as Luis Arraez and Juan Soto maintained elite offensive production despite relatively modest home run totals.
* Offensive success appears to be influenced by multiple factors, including plate discipline, on-base ability, and contact quality.
